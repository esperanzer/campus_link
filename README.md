# campus_link
CampusLink — A university management system built with Laravel, PHP, MySQL, and Tailwind CSS. It provides a clean dashboard for managing students, courses, and lecturers with full CRUD functionality and responsive design.




Features (Planned Modules)


- **Student Management** – Register, view, edit, and track students’ academic details.
- **Course Management** – Assign and organize courses and subjects per department.
- **Department Management** – Manage academic departments and programs.
- **Staff Management** – Handle lecturers, administrators, and support staff.
- **Enrollment & Grades** – Track course enrollment, performance, and transcripts.
- **Attendance Tracking** – Monitor student attendance per subject.
- **User Authentication & Roles** – Secure access for Admins, Staff, and Students.
- **Fees Management (Future)** – Handle student billing and payments.


Tech Stack


- **Language:** PHP
- **Framework:** Laravel
- **Database:** MySQL
- **Frontend:** Blade + Tailwind CSS
- **Version Control:** Git & GitHub
- **Package Manager:** Composer
- **Other Tools:** Laravel Breeze (for authentication)
